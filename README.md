# Onsi tokens sync

This repo is linked to our tokens in figma via [Tokens Studio](https://docs.tokens.studio/) and uses [SD-transforms](https://github.com/tokens-studio/sd-transforms) to transform these tokens into a format [Style Dictionary](https://styledictionary.com/) can manipulate. Style Dictionary is used to compile the various tokens into css and dart code, as definied in build-tokens.js.

# Usage
```
npm i
npm run tokens
```
